# Informator Politechniki Białostockiej

Repozytorium nowego informatora Politechniki Białostockiej.

## Jak zacząć

Ściągnij te repozytorium
```sh
$ git clone https://github.com/jamstackpb/informatorpb
```

Zainstaluj zależoności
```sh
$ npm i
```

## Uruchamianie

```sh
$ npm run dev
```

## Technologie
- markdown
- TypeScript
- ReactJS
- NextJS
